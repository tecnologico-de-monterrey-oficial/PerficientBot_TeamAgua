Before you can access GitHub services using PerficientBot., you need to generate a personal access token. Follow the steps below:

  1. First, go to the GitHub Personal Access Token page.
  2. Be sure to be logged in with your account. You will see a screen similar this:
  3. Click on the Generate new token button, and a dropdown menu will appear. Select the option Generate new token (classic) For general use as shown below:
  4. You may be asked to enter your GitHub password for confirmation. Enter it and proceed.
  5. You will be redirected to the token creation page. Start by providing a description or note to identify the purpose of the token, for example, "PerficientBot" or "Bot Token".
  6. Next, choose an expiration date for the token. You can select a specific date or set it to have no expiration. It is recommended to set an expiration date to ensure better security.
  7. Now, it's time to set the token scopes. Check the following permissions: "repo", "admin:public_key", "admin:org", and "user". These permissions align with the capabilities required by PerficientBot.

    • repo: Grants full control of private repositories.
    • admin:public_key: Provides full control of user public keys.
    • admin:org: Offers full control of organizations and teams, including reading and writing organization projects.
    • user: Allows updating and reading of user data, including user profile information and email addresses.
  8. Once you have selected the necessary scopes, click the Generate token button at the bottom of the page
  9. You will see a confirmation screen indicating that you have successfully created the token. Make sure to click on the copy symbol to copy the token to your clipboard.
    10. This is the only chance you will have to copy it. If you don't, you will have to delete it and start the process again.
  10. Return to PerficientBot > My Services and clear the content of the GitHub Token field. Paste the newly generated token into the field using Ctrl + V.
    11. That's it! You have successfully updated your GitHub access token and can now access GitHub services using PerficientBot.
