Before you can access Outlook services using PerficientBot, you need to obtain an access token from Microsoft Graph API. Follow the steps below:

  1. First, navigate to the Microsoft Graph Explorer website.
  2. Be sure to be logged with your Microsoft Account. You will see a screen similar to the one shown in the reference image below:
  3. Make sure to modify the permissions so that you can create events using PerficientBot. Click on the "Modify Permissions" tab and ensure that "User.Read" and "User.ReadWrite" permissions are active.
  4. Next, switch to the "Access Token" tab and click on the copy symbol to copy the token.
  5. Go back to PerficientBot > My Services, clear the content of the "Outlook Token" field, and paste the newly copied token using Ctrl + V.
  6. That's it! You can now access Outlook services using PerficientBot. You will be able to view and create new events effortlessly.
