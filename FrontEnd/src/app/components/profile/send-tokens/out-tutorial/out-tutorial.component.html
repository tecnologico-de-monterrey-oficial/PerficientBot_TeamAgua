Before you can access Outlook services using PerficientBot., you need to obtain an access token from Microsoft Graph API. Follow the steps below:

  1. First, navigate to the Microsoft Graph Explorer website.
  2. Be sure to be logged with your Perficient Microsoft Account. You will see a screen similar to the one shown in the image below:
  3. Make sure to modify the permissions so that you can create events using PerficientBot. Click on the Modify Permissions. tab
  4. Ensure that User.Read and User.ReadWrite permissions are active, by clicking Consent under Status column.


  5. Next, switch to the Access Token tab and click on the copy symbol to copy the token.


  6. Go back to PerficientBot > My Services, clear the content of the Outlook Token field, and paste the newly copied token using Ctrl + V.


    7. That's it! You can now access Outlook services using PerficientBot.. You will be able to view and create new events effortlessly.
